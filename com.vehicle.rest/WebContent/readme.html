Read Me

This is a How to create a Java Restful Service. Please go to http://www.vehicle.com/

Inventory

To get the version status click:

http://localhost:7001/com.vehicle.rest/api/v1/status/version 

To get the entire database in JSONArray format click:

http://localhost:7001/com.vehicle.rest/api/v1/inventory

To get the specific search pattern choose either of the method:

http://localhost:7001/com.vehicle.rest/api/v2/inventory?reg=<registration_number>

and enter any known vehicle /registration number at the end or the URL. 

or 

http://localhost:7001/com.vehicle.rest/api/v2/inventory/<brand>/<registration_number>

and enter any known vehicle /brand/registration number at the end of the URI to be a little more precise.

or 

http://localhost:7001/com.vehicle.rest/api/v2/inventory/<brand>/<model>/<manufacturing_year>

and enter any known vehicle /brand/model/year at the end of the URI to be a more precise. 

POST

The POST method to insert into the database table. Two different ways to do that:

  • One using data binding (object mapping)
  • Second is to use JSONArrays and JSONObjects

Both ways are valid, choose which one you want to go with. Following link will open the POST web form:

POST web form

PUT

The PUT method very similar to the POST method. Click on the link for a demo of PUT, this web form will allow you to update the registration number number.
